# `client/src/components/`

Reusable components for the client UI.

## Subdirectories

- `screens/`: “phone UI” screens organized by flow (onboarding, conversation, booking, retention, account).
- `shared/`: shared layout and cross-screen components (e.g. PhoneFrame, ChatBubble).
- `ui/`: UI primitives (buttons, dialogs, inputs, etc.), primarily Radix-based.
